声明

JavaScript 有 3 种声明方式：var，let，const

  • var：声明一个变量
  • let：声明一个块作用域的局部变量
  • const：声明一个块作用域的、只读的常量
var x; // 声明一个变量 x，并且没有对 x 进行赋值，未赋值的变量默认值为 undefined

var name = 'Zhang'; // 声明一个变量 name，并赋值为 'Zhang'，此时变量 name 的值为 'Zhange'
name = 'Wang'; // 重新对变量赋值，此时变量 name 的值变成了 'Wang'

if (true) {
  let age = 18; // 声明一个块作用域的局部变量 age，并且赋值为 18，在块作用域内(简单理解就是 {} 括号之间的代码段内)，age 的值为 18
  console.log(age); // 输出结果: 18

  age = 28; // 可以对 age 变量进行重复赋值，此时 age 值为 28
  console.log(age); // 输出结果: 28
}
// 在块作用域外 访问声明的 块作用域变量 会报错
console.log(age); // 输出结果: ReferenceError: age is not defined

const PI = 3.14159; // 声明一个常量(常量就是值不会变化的量)
PI = 6; // 对常量进行重复赋值会报错: TypeError: Assignment to constant variable

数据结构与类型

JavaScript 中有 6 种基本类型，有 1 种复合类型。

基本类型：

  • Boolean：布尔值，true 和 false
  • null：一个表明 null 值的特殊关键字
  • undefined：变量未定义时的属性
  • Number：表示数字，例如 42 3.14159
  • String：表示字符串，例如 'Zhange' "wang"
  • Symbol：ES6 新添加的类型，表示独一无二的值

复合类型：

  • Object：对象
let isRight = true; // 布尔类型
let temp = null; // null 类型
let a; // undefined 类型
let age = 18; // 数值类型
let name = 'Zhang'; // 字符串类型
let s = Symbol(); // symbol 类型
let o = { name: 'Zhange', age: 18 }; // 对象

// 使用 typeof 操作符获取 表达式 的数据类型，返回字符串
console.log(typeof isRight); // 输出: boolean
console.log(typeof temp); // 输出: null
console.log(typeof a); // 输出: undefined
console.log(typeof age); // 输出: number
console.log(typeof name); // 输出: string
console.log(typeof s); // 输出: symbol
console.log(typeof o); // 输出: object
